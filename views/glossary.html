What people say vs what things actually mean inside system execution memory.

01. Trojan Horse "A terrifying hacker file that magically steals your camera access and deletes your hard drive." A piece of compiled software binary using deceptive naming strings to pass file validation parameters, perform process hollowing inside native operating system threads, and initialize an unauthorized local background socket listener port.
02. Network Worm "A dangerous link that jumps out of the web browser and infects every phone on your home internet." A self-replicating background macro requiring zero human execution. It runs local subnet loops scanning IP segments for unpatched open connectivity points (like SMB port 445), exploits memory buffer boundaries, and duplicates its own payload automatically.
03. SQL Injection "A magical code phrase that explodes an entire cloud datacenter network on the dark web." An un-sanitized character argument string that breaks out of developer-intended data encapsulation quotes ('), appends arbitrary relational instruction logic, and forces an interpreter engine to parse unauthorized searches directly inside data pools.
04. Serverless Code "An elite futuristic architecture where software floats in the sky with zero reliance on computer hardware." Static structural asset bundles deployed directly over global edge networks (like GitHub Pages) that shift 100% of their operational execution and rendering weight onto the visitor's local processor and graphics threads for exactly ₹0.00/month in hosting overhead [INDEX].
05. Fileless Threat "An untraceable phantom bug that leaves zero traces and cannot be found by an operating system." A malicious string macro that completely avoids writing permanent file chunks onto local disk drivers, injecting its instructions directly into active volatile RAM memory allocations by leveraging built-in system terminal tools (like PowerShell or Bash scripts).
06. Ransomware "A highly classified military weapon designed by underground syndicates to permanently brick a PC." A basic recursive directory traversal file script that runs high-velocity symmetric encryption loops (like AES-256) over local filesystem pathways to overwrite file bytes, completely neutralized by simple, air-gapped offline data mirrors [INDEX].
07. WebAssembly "A simple frontend configuration tool built to make static web text blocks render slightly faster." A low-level, stack-based binary instruction format compiled directly from Rust or C++ that maps arithmetic operations directly onto native silicon threads, executing at near-hardware speeds right inside an isolated browser memory sandbox [INDEX].
08. Service Worker "A hidden background script built to monitor tracking cookies and log web layouts." A client-side network proxy utility running inside an isolated browser thread that intercepts raw network fetch commands, pulls static components straight from device disk storage cache, and mounts full systems completely offline with 0ms network dependency [INDEX].
09. XSS Attack "A severe server flaw allowing hackers to steal profile passwords across the cloud network." A developer failure to use text-only DOM parsing attributes (like textContent), allowing an incoming input data string to pass raw script structures directly into a viewport layout node where the browser executes it as valid logic.
10. P2P Mesh Net "A complex satellite web framework requiring millions of rupees in infrastructure setups." A direct browser-to-browser UDP data socket communication tunnel established via native WebRTC signaling tokens [INDEX]. Once metadata keys exchange, data streams directly across a local desk network without hitting a centralized external server [INDEX].
11. Rootkit Core "An invisible hacker ghost that permanently lives inside your laptop and cannot be erased by any system software." A highly privileged software suite that modifies kernel or boot configurations (like UEFI firmware). It hooks native OS API functions to manipulate system data outputs—completely hiding its own background processes, files, and active connections.
12. Keylogger "A remote spy program deployed by corporate cartels to record webcam screens and steal identity profiles." A simple, automated event listener hook bound natively to the host operating system's keyboard input driver loops. It captures raw physical key stroke sequences, buffers the text string locally, and pushes them over outbound connection requests.
13. Browser Hijacker "A broken browser glitch that forces pop-up windows open because your laptop is old." An unverified extension that modifies local configuration files (like changing default DNS addresses or proxy routes) to force user browser requests through ad-generating redirect URLs.
14. Logic Bomb "A hidden countdown timer planted by digital saboteurs to make a physical processor blow up." A basic conditional statement (if statement) buried quietly inside a legitimate deployment script tree. It constantly evaluates system checks (like a date, a file deletion event, or an employee ID removal), and triggers a destructive block if conditions match.
15. Botnet Node "A network of compromised supercomputers used by criminal syndicates to shut down global tech platforms." A cluster of internet-connected hardware nodes running a background communication script that silently listens to an IRC channel or a Command and Control (C2) endpoint, awaiting instructions to simultaneously flood target IP addresses (DDoS).
